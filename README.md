# Portfolio Blog

A Full Stack Portfolio & Blog Website built using:

- React
- Spring Boot
- MySQL
- REST APIs

## Features

- Add Projects
- Update Projects
- Delete Projects
- Add Blogs
- Delete Blogs
- REST API Integration
- MySQL Database Storage

## Tech Stack

Frontend:
- React
- Axios
- React Router

Backend:
- Spring Boot
- Spring Data JPA
- Hibernate

Database:
- MySQL
